# VR02 — UAS: Doll House (Wahana Rumah Boneka)

Proyek Ujian Akhir Semester mata kuliah VR (dosen: Calvin Mona Sandehang, M.Sc.).
Soal 2: **Experience Rumah Boneka / Wahana Boneka** — wahana indoor first-person:
player masuk → boarding → naik kereta mini → kereta jalan mengikuti track melewati
display boneka → finish.
